Qu'est-ce que c'est RPGBot?

Un Bot pour aider pour les JDR fait par Henry#6174, avec un inventaire, un marché et une economie, des équipes et des personnages aussi. Chaque utilisateur a un serveur unique un inventaire et une solde bancaire. Les joueurs peuvent vendrent des choses sur le marché que les autres utilisateurs peuvent acheter. Les utilisateurs peuvent créer des personnages avec leur équipe de Pokemon dans des boîtes de rangements. Les serveur administrateurs peuvent ajouter et donner choses au serveur et à ses utilisateurs.

Personnages:

  • allchars Lister tous les personnage de la guilde

  • character Trouver des informations sur un personnage - rp![character|c|char] <nom>

    • create Créer un nouveau personnage rp!character [create|new] <nom>

    Exemple de Personnage:

    Henry: rp!character create Jacques
RPGBot: Décrivez le personnage. (Feuille de personnage pertinente) [Describe the character (Relevant character sheet)]
Henry: Je travaille pour Team Rocket nous voulons voler Pikachu
RPGBot: Quel est le niveau? [What level is the character?]
Henry: 52
RPGBot: Autres informations? Ajoutez un image du personnage utilisant le mot-clé "image". Les formats utilisent une syntaxe regulière par exemple image: http://image.com/image.png, couleur_de_cheveux: brune, surnom: Pierre (Separez les clés avec une virgule ou nouvelle ligne.)
    [Any additional info? (Add a character image using the image keyword. Formats use regular syntax i.e image: http://image.com/, hair_color: blond, nickname: Kevin (Separate keys with commas or newlines)]
Henry: coleur_cheveux: violet, richesse: je suis riche, coleur_des_yeux: bleus
RPGBot: personnage créé! rp!team addmember pour ajouter a l'équipe de votre personnage [Character created! rp!team addmember to add to your characters team!]
    • delete Supprimer un personnage du nom donné (Vous devez être le propriétaire) rp!character [delete|remove] <nom>

  • characters Lister tous vos personnages rp![characters|chars]

Economie:

  • bid Placer une offre sur l'enchère actuelle dans le channel.
  • economy Voir le solde bancaire de quelqu'un rp![economy|bal|balance|eco|e] [membre]
    • givemoney Donner l'argent aux membres (Modérateurs) rp!economy [givemoney|give] <montant> [membres...]
    • setbalance Fixer le solde bancaire des membres à un montant rp!economy [setbalance|set] <montant> [membres...]
  • lootbox Lister les lootboxes actuelles rp![lootbox|lb]
    • buy Acheter un lootbox rp!lootbox buy <nom>
    • create Créer un nouveau lootbox.
      Utilisez {chose}x{#} notation pour ajouter les choses avec {#} poids.
      Poids est un entier. Pour exemple avec
      bananex2 orangex3: Le resultat de la boîte sera
      un choix au hasard de [banane, banane, orange, orange, orange]
      rp!lootbox [create|new] <nom> <coût> [choses...]
    • delete Supprimer une lootbox rp!lootbox [delete|remove] <nom>
  • lotto Lister les loteries actuelles. rp![lotto|lottery]
    • enter Rejoindre la loterie. rp!lotto [enter|join] <nom>
    • new Créer une nouvelle loterie, avec une cagnotte fixé par un temps en secondes rp!lotto [new|create] <nom> <cagnotte> <temps>
  • market Voir le marché des joueurs rp![market|m|pm]
    • buy Acheter un objet du marché avec l'ID donné rp!market [buy|purchase] <id>
    • create Vendre quelque chose sur le marché rp!market [create|createlisting|new|listitem|list] <coût> <quantité> <chose>
    • search Chercher un objet sur le marché rp!market search <objet>
  • pay Payer à quelqu'un un montant rp!pay <montant> <membre>
  • shop Lister tous les objets sur le magasin du serveur

    • additem Ajouter une chose au magasin du serveur, pour rendre un objet invendable ou non-achetable fixer leur valeurs respectives à 0

      
  Henry: rp!shop additem Pokéball
  RPGBot: Dites 'cancel' pour annuler ou 'skip' pour skip. A quel prix doit il être achetable? 0 pour pas achetable. [Say 'cancel' to cancel or 'skip' to skip a step. How much should this be buyable for? 0 for not buyable]
  Henry: 0
  RPGBot: Quel sera son prix de vente? 0 pour pas vendable
  Henry: 10
  RPGBot: Quel doit être le niveau minimum d'un utilisateur pour l'acheter? 0 pour pas minimum
  Henry: 0
  RPGBot: Magasin actualisé [Guild shop updated]

      Peut être vendu pour $10 et ne peut pas être acheté. Utilisateur ne doit pas avoir un niveau minimum. Il faut que sa soit un objet qui existe! Cela demande le grade Bot Moderator ou Admin rp!shop [additem|add] <nom>

    • buy Acheter quelque chose dans le magasin rp!shop buy <chose> <montant>

    • removeitem Supprimer quelque chose du magasin. Cela demande le grade Bot Mod or Bot Admin rp!shop removeitem <nom>

    • sell Vendre quelque chose au magasin rp!shop sell <chose> <montant>

  • startbid Commencer un enchère pour quelque chose rp!startbid <chose> <montant> <enchère commencant>

Groupes:

  • guild Trouver information sur la guilde d'un utilisateur. Subcommandes pour la gestion d'une guilde

    • create Créer un guilde nouvelle rp!guild create <nom>

      Exemple d'Utilisation:

      
  Henry: rp!guild create Team Rocket
  RPGBot: Disez 'cancel' pour annuler ou 'skip' pour passer. Décrivez la guilde ['cancel' or 'skip' to cancel creation or skip a step. Describe the Guild (guild description)]
  Henry: Nous voulons voler Pikachu et nous sommes cool
  RPGBot: Cette guilde est-elle ouverte à tous? Ou a-t-on besoin d'une invitation (yes [oui] ou no [non]) [Is this guild open to everyone? Or is an invite necessary? (yes or no, no is assumed)]
  Henry: yes [oui]
  RPGBot: Si vous voulez, donnez l'URL d'une image pour la guilde [If you'd like give a URL to an image for the guild]
  Henry: http://cdn.bulbagarden.net/upload/thumb/d/d3/Team_Rocket_anime.png/300px-Team_Rocket_anime.png
  RPGBot: Finalment vous pouvez aussi fixer une icône pour la guilde [Finally, you can also set an icon for the guild]
  Henry: https://www.seti.soton.ac.uk/static/uploads/socialgroupicon_36_1265776899.gif
  RPGBot: Guilde créée! [Guild successfully created!]

    • delete Supprimer votre guilde

    • deposit Deposer une somme d'argent dans la banque de votre guilde rp!guild deposit <montant>

    • deposititems Deposer des objets dans la baque de votre guilde, utilisez {objet}x{#} notation (i.e. bananax3) rp!guild deposititems [items...]

    • info Obtenez les informations sur une guilde rp!guild info <name>

    • invite Invitez un utilisateur dans votre guilde fermée rp!guild invite <user>

    • join Rejoindre une guilde (si vous avez unz invitation pour une guilde) rp!guild join <name>

    • kick Virez quelqu'un de votre guilde rp!guild kick <user>

    • leave Quitter votre guilde

    • setdescription Ecrire la description de la guilde rp!guild [setdescription|setdesc] <description>

    • seticon Définir l'icône de la guilde rp!guild seticon <url>

    • setimage Définir l'image de la guilde rp!guild setimage <url>

    • setmod Donne le rang modérateur aux personnes listées (meneur de la guilde seulement) rp!guild setmod [membres...]

    • toggleopen Change l'etat de la guilde a "ouvert" ou "fermé"

    • withdraw Retirer argent de la banque de la guilde rp!guild withdraw <montant>

    • withdrawitems Retirer des objets de la banque de la guilde (modérateurs de la guilde seulement, même syntaxe que pour déposer des items) rp!guild withdrawitems [objets...]

  • guilds Liste les guildes

Inventaire:

  • inventory Regarder votre inventaire ou celui d'un autre. rp![inventory|i|inv] [membre]
    • give Donner des objets ({item}x{#}) à un membre; ie: rp!give @Henry#6174 pokeballx3 rp!inventory give <autre> [objets...]
    • giveitem Donner un objet a un utilisateur (Pas depuis votre inventaire, vous devez être modérateur) rp!inventory giveitem <item> <num> [members...]
    • takeitem Supprimer un objet de l'inventaire d'une personne (Devez être modérateur) rp!inventory [takeitem|take] <item> <num> [members...]

Divers:

  • donate Informations pour les dons

  • feedback Donnez moi des retours sur le bot rp!feedback <feedback>

  • info Infos sur le bot

  • ping Tester le ping du bot

  • rtd Lancer un dé un certain nombre de fois et avec un certain nombre de faces (ndx notation)
    Exemple: rp!rtd 3d7 2d4
    Ajouts Optionnels:
    Vérifier la réussite en mettant un >/<#

    Obtener le meilleur lancer possible en mettant un ^n

    Ajouter des nombres au résultat final (pos ou neg)

    Exemples de tout:

    Henry: rp!rtd 8d8 -12 15 ^4 >32
RPGBot: Roll failed (30 > 32) ([8 + 7 + 6 + 6] + -12 + 15) (Grabbed top 4 out of 8)

    rp![rtd|rollthedice|dice] [dice...]

  • source Displays my full source code or for a specific command.
    To display the source code of a subcommand you have to separate it by
    periods, e.g. tag.create for the create subcommand of the tag command.
    rp!source [command]

  • totalcmds Get totals of commands and their number of uses

Pokemon:

  • box Check the pokemon in your box rp!box [member]
  • pokemon Subcommands for Pokemon management, see rp!help pokemon. Same use as rp!box rp![pokemon|p] [member]

    • create Create a new Pokemon to add to your box rp!pokemon [create|new]

      Example Usage:

        Henry: rp!pokemon create
  RPGBot: In any step type 'cancel' to cancel. What will its nickname be?
  Henry: Charry
  RPGBot: What species of Pokemon is it?
  Henry: Charizard
  RPGBot: In any order, what are its stats? (level, health, attack, defense, spatk, spdef)For example level: 5, health: 22, attack: 56 Type 'skip' to skip.
  Henry: level: 23, health: 42, attack: 52, spdef: 12, health: 88, spatk: 88
  RPGBot: Any additional data? (Format like the above, for example nature: hasty, color: brown)
  Henry: nature: adamant, color: shiny
  RPGBot: Finished! Pokemon has been added to box with ID 1

    • info Get info on a Pokemon rp!pokemon info <id>

    • trade Offer a trade to a user.
      your_id is the ID of the Pokemon you want to give, their_id is the Pokemon you want from them.
      other being the user you want to trade with rp!pokemon trade <your_id> <their_id> <other>

    • release Release a Pokemon from your box rp!pokemon [release|delete|rm|remove] <id>

Settings:

  • settings Get the current server settings rp![settings|s|configuration|conf]

    • additem Add a custom item rp!settings additem <name>

    Example Usage:

    Henry: rp!s additem Banana
RPGBot: Describe the item (a description for the item)
Henry: A delicious yellow fruit
RPGBot: Additional information? (Attributes formatted in a list i.e `color: 400, value: 200`)
Henry: color: yellow, taste: delicious, type: fruit
RPGBot: Item successfully created
    • iteminfo Get info on a server item rp!settings iteminfo <item>
    • items See all items for a guild

Team:

  • team Check a character’s team rp!team <character>
    • add Add a Pokemon to a character’s team rp!team [add|addmember] <character> <id>
    • remove Remove a Pokemon from a character’s team rp!team [remove|removemember] <character> <id>

User:

  • userinfo Get info on a user rp![userinfo|ui] [user]
  • experience Get your or another user’s level information. Help on this command for experience subcommands
    EXP is calculated using a 0.1x^2+5x+4 where x is equal to the user’s current level
    Spamming commands or messages will not earn more exp! rp![experience|exp] [member]
    • add Give the given members an amount of experience rp!experience add <amount> [members...]
    • setlevel Set the given members level rp!experience setlevel <level> [members...]
