Qu'est-ce que c'est RPGBot?

Un Bot pour aider pour les JDR fait par Henry#6174, avec un inventaire, un marché et une economie, des équipes et des personnages aussi. Chaque utilisateur a un serveur unique un inventaire et une solde bancaire. Les joueurs peuvent vendrent des choses sur le marché que les autres utilisateurs peuvent acheter. Les utilisateurs peuvent créer des personnages avec leur équipe de Pokemon dans des boîtes de rangements. Les serveur administrateurs peuvent ajouter et donner choses au serveur et à ses utilisateurs.

Personnages:

  • allchars Lister tous les personnage de la guilde

  • character Trouver des informations sur un personnage - rp![character|c|char] <nom>

    • create Créer un nouveau personnage rp!character [create|new] <nom>

    Exemple de Personnage:

    Henry: rp!character create Jacques
RPGBot: Décrivez le personnage. (Feuille de personnage pertinente) [Describe the character (Relevant character sheet)]
Henry: Je travaille a Team Rocket nous voulons voler Pikachu
RPGBot: Quel est le niveau? [What level is the character?]
Henry: 52
RPGBot: Autre information? Ajoutez un image du personnage utilisant le mot-clé "image". Formats utilisent syntaxe regulier par exemple image: http://image.com/image.png, coleur_de_cheveux: brune, surnom: Pierre (Separez clés avec virgule ou nouvelle ligne.)
    [Any additional info? (Add a character image using the image keyword. Formats use regular syntax i.e image: http://image.com/, hair_color: blond, nickname: Kevin (Separate keys with commas or newlines)]
Henry: coleur_cheveux: violet, richesse: je suis riche, coleur_des_yeux: bleus
RPGBot: personnage créé! rp!team addmember pour ajouter au équipe de votre personnage [Character created! rp!team addmember to add to your characters team!]
    • delete Supprimer un personnage du nom donné (Vous devez être le propriétaire) rp!character [delete|remove] <nom>

  • characters Lister tous vos personnage rp![characters|chars]

Economie:

  • bid Placer une offre sur la chose actuelle dans le channel.
  • economy Voir le solde bancaire de quelqu'unrp![economy|bal|balance|eco|e] [membre]
    • givemoney Donner les membres argent (Modérateurs) rp!economy [givemoney|give] <montant> [membres...]
    • setbalance Fixer le solde bancaire des membres à un montantrp!economy [setbalance|set] <montant> [membres...]
  • lootbox Lister les lootboxes actuels rp![lootbox|lb]
    • buy Acheter un lootboxrp!lootbox buy <nom>
    • create Créer un nouveau lootbox.
      Utilisez {chose}x{#} notation pour ajouter les choses avec {#} poids.
      Poids est un entier. Pour exemple avec
      bananex2 orangex3: Le resultat du boîte sera
      un choix au hasard de [banane, banane, orange, orange, orange]
      rp!lootbox [create|new] <nom> <coût> [choses...]
    • delete Supprimer un lootboxrp!lootbox [delete|remove] <nom>
  • lotto Lister les loteries actuels. rp![lotto|lottery]
    • enter Entrer la loterie. rp!lotto [enter|join] <nom>
    • new Créer un nouvelle lotto, avec un cagnotte fixé pendant le temps en secondesrp!lotto [new|create] <nom> <cagnotte> <temps>
  • market Voir le marché des joueursrp![market|m|pm]
    • buy Acheter une chose du marché avec l'ID donnérp!market [buy|purchase] <id>
    • create Vendre quelque chose sur le marché rp!market [create|createlisting|new|listitem|list] <coût> <montant> <chose>
    • search Chercher une chose sur le marché rp!market search <chose>
  • pay Payer quelqu'un un montantrp!pay <montant> <membre>
  • shop Lister toutes les choses sur le magasin du serveur

    • additem Ajouter une chose à le magasin du serveur, pour fair une chose invendable ou pas achetable fixer leur valeurs respectifs à 0

      
  Henry: rp!shop additem Pokéball
  RPGBot: Disez 'cancel' pour annuler ou 'skip' pour passer. Combien il devra être achetable pour? 0 pour pas achetable. [Say 'cancel' to cancel or 'skip' to skip a step. How much should this be buyable for? 0 for not buyable]
  Henry: 0
  RPGBot: Combien il devra être vendable pour? 0 pour pas vendable
  Henry: 10
  RPGBot: Quel est le niveau minimum un utilisateur doit être pour cette chose? 0 pour pas minimum
  Henry: 0
  RPGBot: Magasin actualisé [Guild shop updated]

      Peut être vendu pour $10 et ne peut pas être acheté. Utilisateur ne doit pas être un niveau certain. Il faut être une chose qui existe! Cela demande Bot Moderator or Admin rp!shop [additem|add] <nom>

    • buy Acheter quelque chose du magasin rp!shop buy <chose> <montant>

    • removeitem Supprimer quelque chose du magasin. Cela demande Bot Mod or Bot Admin rp!shop removeitem <nom>

    • sell Vendre quelque chose au magasin rp!shop sell <chose> <montant>

  • startbid Commencer un enchère pour quelque chose rp!startbid <chose> <montant> <enchère commencant>

Groupes:

  • guild Trouver information sur la guilde d'un utilisateur. Subcommandes pour la gestion d'une guilde

    • create Créer un guilde nouvelle rp!guild create <nom>

      Exemple d'Utilisation:

      
  Henry: rp!guild create Team Rocket
  RPGBot: Disez 'cancel' pour annuler ou 'skip' pour passer. Décrivez la guilde ['cancel' or 'skip' to cancel creation or skip a step. Describe the Guild (guild description)]
  Henry: Nous voulons voler Pikachu et nous sommes cool
  RPGBot: Cette guilde est-elle ouverte à tous? Ou on a besoin d'une invitation (yes [oui] ou no [non]) [Is this guild open to everyone? Or is an invite necessary? (yes or no, no is assumed)]
  Henry: yes [oui]
  RPGBot: Si vous voulez, donnez l'URL d'une image pour la guilde [If you'd like give a URL to an image for the guild]
  Henry: http://cdn.bulbagarden.net/upload/thumb/d/d3/Team_Rocket_anime.png/300px-Team_Rocket_anime.png
  RPGBot: Finalment vous pouvez aussi fixé une icône pour la guilde [Finally, you can also set an icon for the guild]
  Henry: https://www.seti.soton.ac.uk/static/uploads/socialgroupicon_36_1265776899.gif
  RPGBot: Guilde créé! [Guild successfully created!]

    • delete Delete your guild

    • deposit Deposit an amount of money into the guild bank rp!guild deposit <amount>

    • deposititems Deposit items into the guild’s storage, uses {item}x{#} notation (i.e. bananax3) rp!guild deposititems [items...]

    • info Get info on a guild rp!guild info <name>

    • invite Invite a user your closed guild rp!guild invite <user>

    • join Join a guild (if you have an invite for closed guilds) rp!guild join <name>

    • kick Kick a member from a guild rp!guild kick <user>

    • leave Leave your guild

    • setdescription Set the guild’s description rp!guild [setdescription|setdesc] <description>

    • seticon Set the guild’s icon rp!guild seticon <url>

    • setimage Set the guild’s image rp!guild setimage <url>

    • setmod Give the listed users mod for your guild (guild owner only) rp!guild setmod [members...]

    • toggleopen Toggle the Guilds open state

    • withdraw Take money from the guild bank rp!guild withdraw <amount>

    • withdrawitems Withdraw items from the guild (guild mods only, same syntax as deposit items) rp!guild withdrawitems [items...]

  • guilds List guilds

Inventory:

  • inventory Check your or another users inventory. rp![inventory|i|inv] [member]
    • give Give items ({item}x{#}) to a member; ie: ;give @Henry#6174 pokeballx3 rp!inventory give <other> [items...]
    • giveitem Give an item to a person (Not out of your inventory, must be Bot Moderator) rp!inventory giveitem <item> <num> [members...]
    • takeitem Remove an item from a person’s inventory (Must be Bot Moderator) rp!inventory [takeitem|take] <item> <num> [members...]

Misc:

  • donate Donation information

  • feedback Give me some feedback on the bot rp!feedback <feedback>

  • info Bot Info

  • ping Test the bot’s connection ping

  • rtd Roll a number of dice with given sides (ndx notation)
    Example: rp!rtd 3d7 2d4
    Optional Additions:
    Test for success by adding a >/<#

    Grab the top n rolls by adding ^n

    Add to the final roll by just adding a number (pos or neg)

    Examples of all:

    Henry: rp!rtd 8d8 -12 15 ^4 >32
RPGBot: Roll failed (30 > 32) ([8 + 7 + 6 + 6] + -12 + 15) (Grabbed top 4 out of 8)

    rp![rtd|rollthedice|dice] [dice...]

  • source Displays my full source code or for a specific command.
    To display the source code of a subcommand you have to separate it by
    periods, e.g. tag.create for the create subcommand of the tag command.
    rp!source [command]

  • totalcmds Get totals of commands and their number of uses

Pokemon:

  • box Check the pokemon in your box rp!box [member]
  • pokemon Subcommands for Pokemon management, see rp!help pokemon. Same use as rp!box rp![pokemon|p] [member]

    • create Create a new Pokemon to add to your box rp!pokemon [create|new]

      Example Usage:

        Henry: rp!pokemon create
  RPGBot: In any step type 'cancel' to cancel. What will its nickname be?
  Henry: Charry
  RPGBot: What species of Pokemon is it?
  Henry: Charizard
  RPGBot: In any order, what are its stats? (level, health, attack, defense, spatk, spdef)For example level: 5, health: 22, attack: 56 Type 'skip' to skip.
  Henry: level: 23, health: 42, attack: 52, spdef: 12, health: 88, spatk: 88
  RPGBot: Any additional data? (Format like the above, for example nature: hasty, color: brown)
  Henry: nature: adamant, color: shiny
  RPGBot: Finished! Pokemon has been added to box with ID 1

    • info Get info on a Pokemon rp!pokemon info <id>

    • trade Offer a trade to a user.
      your_id is the ID of the Pokemon you want to give, their_id is the Pokemon you want from them.
      other being the user you want to trade with rp!pokemon trade <your_id> <their_id> <other>

    • release Release a Pokemon from your box rp!pokemon [release|delete|rm|remove] <id>

User:

  • userinfo Get info on a user rp![userinfo|ui] [user]
  • experience Get your or another user’s level information. Help on this command for experience subcommands
    EXP is calculated using a 0.1x^2+5x+4 where x is equal to the user’s current level
    Spamming commands or messages will not earn more exp! rp![experience|exp] [member]
    • add Give the given members an amount of experience rp!experience add <amount> [members...]
    • setlevel Set the given members level rp!experience setlevel <level> [members...]
